    OverlayHeader

    Simple header component that displays a responsive logo an avatar and a close button.

    <OverlayHeader
      onClose={() => alert('Close clicked')}
      avatar={{ url: 'https://github.com/transferwise.png' }}
      profileType={OverlayHeader.ProfileType.PERSONAL}
      theme={OverlayHeader.Theme.LIGHT}
    />;
    TypeError: Cannot read property 'PERSONAL' of undefined
