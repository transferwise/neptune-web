## Getting started

To use these classes you need to include the `neptune-social-media.css` file after your import of `neptune.css`:

```js
import '@transferwise/neptune-css/dist/css/neptune.css';
import '@transferwise/neptune-css/dist/css/neptune-social-media.css';
```

## Usage

Simply apply the correct class to a button along with the `btn` class, as follows:

```html
<button type="button" className="btn btn-google">Login with Google</button>
<button type="button" className="btn btn-facebook">Login with Facebook</button>
```
