## History

These CSS components are based on a **[fork of Bootstrap v3.3.4](https://github.com/transferwise/bootstrap)**, from 2015.
Since then we have changed, customised, and created new components according to the TransferWise theme.

In early 2019 Neptune was born, and it became the perfect time to evolve the codebase from a monolith to serving individual modules that are more compliant to standard CSS.```

## What Neptune CSS distributes

In the dist directory you can find bundles for the individual components, variables, fonts, and other common utilities.

Use the **CSS bundles as individual** components or import the **whole library**.

```html
neptune-css dist/
├── css/
│ └── variables
│   ├── _colors.less
│   ├── _custom-media.less
│   ├── _sizes.less
│   └── ...
│ │
│ ├── accordion.css
│ ├── alerts.css
│ ├── background.css
│ ├── badge.css
│ ├── ...
│ └── neptune.css
│
└── fonts/
│ ├── glyphicons-halflings-regular.eot
│ ├── ...
│ ├── TransferWise-IconFont.eot
│ ├── ...
│ ├── TW-Averta-*.eot
│ └── ...
│
└── img/
 ├── *.svg
 └── *.png

```
